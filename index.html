Sobre o Projeto

Este site faz parte de um teste técnico de estágio DevOps. O objetivo é provisionar um site estático em nuvem pública com uma esteira CI/CD completa, desde o commit até o deploy automatizado.

Stack Utilizada

☁️ AWS S3

Armazenamento e hospedagem do site estático com alta disponibilidade.

🌐 CloudFront

CDN global da AWS para distribuição de conteúdo com baixa latência.

🔄 GitHub Actions

Pipeline CI/CD automatizada disparada a cada push na branch main.

🔍 SonarCloud

Análise de qualidade e segurança do código de forma contínua.
